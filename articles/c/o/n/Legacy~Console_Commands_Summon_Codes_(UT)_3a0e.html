Legacy:Console Commands/Summon Codes (UT)

From Unreal Wiki, The Unreal Engine Documentation Site

< Legacy:Console Commands

You need to be logged in as Admin or playing offline to use the Summon Command.

The package and class names specified with the command are case sensitive.

Not all objects are "summonable". If an object fails to appear check your game log file for error messages.
