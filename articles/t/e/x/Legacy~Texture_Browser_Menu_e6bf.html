Legacy:Texture Browser Menu

From Unreal Wiki, The Unreal Engine Documentation Site

This page is the reference for the menubar in the Texture Browser window.

See UnrealEd Menus for other menus.

File

New 
Allows you to create a FractalTexture or WaterTexture using UT's particle system.
Open 
Open texture packages. The browser should automatically goto UT\Textures directory. There should be a ton of files in here. Say your maps is a castle themed map. So navigate to NaliCast.utx and open. You should see a whole bunch of new textures now. Plus the texture pack name should reflect NaliCast now.
Save 
Allows you to save a compiled texture package. Note: do not save the packages that come with UT.
Import 
Allows you to import a 256-color, power-of-2, .pcx image for use as a texture. See Texture Import And Export.
Export
Allows you to export a texture from a package.

Edit

Properties 
Opens up a selected texture's properties window. Option allow assignment of detail and macro textures, animation frames, and drawscale. See Texture Properties.
Delete
Deletes selected texture.
Rename
Allows you to rename the selected texture.
Prev/Next Group 
Scrolls through sub-groups within a package.

View

SetSize 
change the viewing size of the textures displayed.

Do keep in mind that changing the display size does not affect your textures in the game world.

Variable 
Docked 
dock or undock this window from the master browser
