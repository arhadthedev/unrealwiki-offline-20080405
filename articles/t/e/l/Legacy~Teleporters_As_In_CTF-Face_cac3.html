Legacy:Teleporters As In CTF-Face

From Unreal Wiki, The Unreal Engine Documentation Site

Assuming that you've already read the Teleportation Basics tutorial, here's how to get the visual effect seen in UT's CTF-Face. Do as in the basic teleporter tutorial and come back here.

Select the Teleporter's actor properties and set:

Advanced -> bHidden True
Display -> bUnlit True
Display -> DrawType DT_Sprite
Display -> Skin [[Texture Topics pick a texture]] you like from SpaceFX.utx â€“ in the Texture Browser, open SpaceFX.utx and select. ZxAnPhOrIaN: Copy and paste this visible teleporter into UED2.0 (the editor that came with the original UT). Then play the level when you have a player start actor in the map. You will see a swirl-thing around the man-with-arrow texuture around it.

Begin Map
Begin Actor Class=VisibleTeleporter Name=VisibleTeleporter0
    bDynamicLight=True
    Level=LevelInfo'MyLevel.LevelInfo0'
    Tag=VisibleTeleporter
    Region=(Zone=LevelInfo'MyLevel.LevelInfo0',ZoneNumber=1)
    Location=(X=112.000000,Y=-48.000000,Z=-80.000000)
    OldLocation=(X=112.000000,Y=-48.000000,Z=-80.000000)
    bSelected=True
    DrawType=DT_Sprite
    Style=STY_Normal
    Mesh=None
    Name=VisibleTeleporter0
End Actor
End Map
