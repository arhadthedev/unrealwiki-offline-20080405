Open Source

The Unreal Wiki is a helpful resource for budding mappers and scripters offering a variety of tutorials and a wide range of subjects. In addition to this, we offer here a variety of different open source scripts and classes that perform specific functions. These functions could range from a simple map actor that sets a player who touches it on fire, or a coding-specific array sort algorithm. For the most part, these items can be used in a relatively unhindered fashion but take note of the authors wishes. See Wiki:OpenSource for more details on what 'Open Source' entails.

There are a lot of misconceptions about Open Source going around. One of them is that Open Source would be free of copyright; it's not, and far from it. Every bit of Open Source software is copyrighted and comes with a license. Sometimes that license says that you can use the code at will. In any case, you're bound to adhere to whatever licensing terms come with the code. The individual original authors of the samples listed below might have different stances on that, so read the accompanying statements carefully.

There are a few ways to use these classes and scripts in your own map. Your options are:

  • Compiled files. You can re-distribute them with your map, or you can import the classes you need: see Embedding Code.
  • Classes embedded in a map file.
  • Text scripts which you have to paste into a map yourself (see Create A Subclass).

Scripts

Mapping-Specific

  • Another Pawn - A generic pawn that can be spawned using a Pawn Factory.
  • Incremental Trigger - A trigger that will fire an event out of a series each time it is triggered.
  • Pawn Factory - A generic factory for creating pawns.
  • Random Trigger - A custom trigger that randomly fires off events from a list of event names.
  • Round Robin - A map actor that lets you trigger a series of events.
  • Score Award Trigger - A trigger that, when triggered, increases a player's score.
  • TeamVehicleFactory - A generic factory for use in any vehicle gametype that allows controls that range from team and timing to firing events and random vehicles.
  • Variable Timed Mover - A mover that allows the level designer to set a different MoveTime for each mover key set.

Coding-Specific

These classes/scripts are intended for coding projects that don't neccesarilly rely on specific maps. To use them, you will need to either compile them using the Unreal Editor (not recommended) or by using UCC.

  • Reloadable Weapon - A custom base class that adds support for reloadable weapons.
