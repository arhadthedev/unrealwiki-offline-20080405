Legacy:UClasses

From Unreal Wiki, The Unreal Engine Documentation Site

UClasses's main functionality is being a stand-alone Unreal classes browser with full-text search capabilities (something UnrealEd's built-in actor browser lacks). In addition, it provides functionality like UnrealEd for adding new classes, compiling packages and so on.
