Legacy:UnrealEngineRegFix

From Unreal Wiki, The Unreal Engine Documentation Site

UnrealEngine Registry Fixer is an small utility to restore Windows Registry issues of Unreal Engine based games and is created by Jan "eGo" Urbansky (Jan). It is a plugin based utility and can be expand with more plugins. "UnrealEngine Registry Fixer" is the successor of my well-known and very old tool called "UT99 Registry Fixer".

BETA 1 currently available
