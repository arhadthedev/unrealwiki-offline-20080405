New Effect: Security Mirror:

So far its only being used by the local machine, without server code so it doesn't create any lag for other players - but then they also don't see your reflection - only their's.

It's my 'Security Mirror', which is on a curved surface - and currently only shows players in 'behindview' mode since the engine doesn't draw the player model locally while in FP mode. I have managed to get the image to display correctly, I just haven't replaced the images here yet.

The only shapes I'm using so far are from spheres:
1) half/quarter..for a corner
2) quarter.......a hallway
3) half..........& a ceiling

I also have plans to use other shapes like the shield on the shield gun for a wide area effect.

Since these mirrors can be scoped in on to view activity in otherwise hidden areas from the player they can be aware of other players in that area that they wouldn't normally be able to see.

I'm going to use my effect for a number other uses like, realistic crystals, glassware, and other optical effects.
