Legacy:Animation Browser

From Unreal Wiki, The Unreal Engine Documentation Site

The Animation Browser is one of several Resource Browsers in the UnrealEd Interface. By default it is docked in the Master Browser.

Uses of the browser:

  • displays currently loaded skeletal meshes
  • allows further packages to be opened
  • allows properties of the skeletal mesh resources to be edited, such as skins or animation sequences. But remember Do not alter default packages.

Opening

The Animations Browser is accessed either of two ways:

  • Do UnrealEd Main Menu -> View -> Animations Browser
  • Press the Animations Browser button in the middle of the toolbar at the top of the Editor window. It looks like an "A".

Browser Toolbar Buttons

The Animation Browser has the following buttons (from left to right) in its toolbar:

  • Dock / undock from browser window
  • Open Package
  • Save Package
  • Load Entire Package
  • Import Mesh
  • Import Skeletal Animation
  • Import additional animation data
  • Link animation to mesh
  • Info
  • Edit mesh properties
  • Edit animation properties
  • View bounds
  • View bones
  • Clear groups
  • View reference pose
  • View influences
  • Toggle raw offset display
  • Redigest raw data
  • Copy mesh properties
  • Pase mesh properties
  • Cycle through the LOD mesh levels
  • Import a single LOD mesh
  • Redigest LOD levels
  • Forced syncronous in-level animation
