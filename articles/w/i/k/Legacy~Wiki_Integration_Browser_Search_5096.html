Mozilla and Firefox keyword search

In Mozilla, Wikipedia:Firefox, and other Gecko browsers, add the following URL as a bookmark and set a keyword (such as "uw" or "unreal") to perform quick searches in the address bar:

  • http://wiki.beyondunreal.com/wiki?search=%s

Sidebar Search

Mozilla & Firefox



Conveniently search the Unreal Wiki from Mozilla's search sidebar or Firefox's toolbar search.

Install

  1. Go to the Mycroft plug-in directory
  2. Click on the plug-in name to install it.
  • Netscape 7 (maybe other versions too) won't show more then four search plugins when in simple mode.
  • May not show sidebar results unless sidebar is used for search.

Uninstall

Remove unrealwiki.src from the searchplugins subfolder of your Mozilla program folder.

Alternatively, you could make your own searchplugin using the following code as a guide:

<search
 version="7.1"
 name="UnrealWiki Search"
 description="UnrealWiki - Search the UnrealWiki website." 
 action="http://wiki.beyondunreal.com/wiki"
 searchForm="http://wiki.beyondunreal.com/wiki"
 method="GET"
 >

<input name="search" user>


<interpret 
 resultListStart="<!– RESULT LIST START –>"
 resultListEnd="<!– RESULT LIST END –>"
 resultItemStart="<!– RESULT ITEM START –>"
 resultItemEnd="<!– RESULT ITEM END –>"
{{innerbox|
}}
</search>
<browser
 update="http://mycroft.mozdev.org/update.php/id0/unrealwiki.src" 
 updateIcon="http://mycroft.mozdev.org/update.php/id0/unrealwiki.png" 
 updateCheckDays="7" 
{{innerbox|
}}

Simply save that as a .src file. Place it in your searchplugins folder, and within that same folder, include a 16x16 PNG file for the icon. (Make sure the PNG is named the same as the *.src

Note that the browser tag is only there for mycroft.org submission. You don't really need it.

Opera

With just a bit of manual work, you can neatly integrate Unreal Wiki's search feature into Opera's other search features.

  1. Find search.ini in your Opera program folder and open it in Notepad (or another text editor of your choice).
  1. Scroll to the end of the file and append the following lines to it: 
      [Search Engine 8]

 Name=Unreal &Wiki
 URL=http://wiki.beyondunreal.com/wiki?search=%s

Key=w

  1. Save search.ini and restart Opera.

Note that this overrides the standard search number 8 defined in search.ini. If that is one you need, choose a different number. Unfortunately it's not possible to get both a new dedicated Unreal Wiki search box and keep all of the old ones; but it stands to reason that nobody uses all of them anyway. If you don't need or want the dedicated Unreal Wiki search box and are content with the other features, simply give the Unreal Wiki search engine a unique new number in search.ini.

To display the dedicated Unreal Wiki search box in the Personal Bar, enable it in the Show searches submenu of the Personal Bar's right-click menu. (The Unreal Wiki search is also available from the Search with submenu when you double-click on a term on a web page.)

Opera 9+

Opera 9 and above has the ability to edit the search engine list directly from the Tools->Preferences menu. Just click Tools, Preferences, and open the Search tab. Press Add, enter UnrealWiki for the name, enter a shortcut, and use http://wiki.beyondunreal.com/wiki?search=%s for the URL.
