Info

  • action=version - show program-version (and options ?)
  • action=index - Show all pages
  • action=links - Show all links. cf. MeatBall:LinkDatabase for documentation of various options.
  • action=rc - Show recent changes.

User

  • action=login - Login as known user
  • action=newlogin - Login as new user
  • action=editprefs - Edit user-preferences

Page

The following links use this page for example, but id can equal any page.

  • action=browse&id=Wiki_System_Commands – Browse a page
  • action=edit&id=Wiki_System_Commands – Edit a page
  • action=history&id=Wiki_System_Commands – History of a page

Search

  • search=some+text – Search for some text
