Description

It bothers me that there is no instagib mode for Onslaught. However, the reasons for disallowing an IG rifle are valid. This mutator attempts to create a balanced instagib style game while conceding to the gametypes balance requirements.

Essentially this means that there is no true IG mode for Onslaught. However it may be possible to get closer than it is already. The following rules will apply.

  • The link gun will be tweaked such that primary fire is an IG beam against people and a normal Shock Rifle against everything else.
  • The link gun secondary fire will remain the same - both im terms of function and damage.
  • The vehicle's weaponry will remain unchanged.
  • All players will start with an Avril and some ammo. Probably two - three rockets.

Given that standard IG rifles have infinite ammo I'm left with a choice. Do I keep the ammo limitation on the IG element (link primary fire), or, do I give the players infinite ammo for all the weapons they are carrying. Tis an interesting one. At the moment I'm favouring ammo limitations.
