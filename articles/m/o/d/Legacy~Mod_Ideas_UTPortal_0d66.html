Description

This would be a single-player mod for Unreal Tournament that is something like VALVe Software's game 'Portal'. It would include all the same kinds of puzzles, sentryguns, and the notorious cubes. I won't make exact replicas of any Portal components for fear of copyright issues.

Plot

The game starts similarly to Portal, with the main character running an obstacle course that must be solved with the portal gun. You start the game with the 'gravity gun', similar to the one in HL2, which allows you to pick up cubes and complete puzzles, in addition to using the Portal gun, which you are given shortly after the beginning of the game. There are actually two portal guns in the game: one that is like the original Portal gun, and another, which I scripted myself, which fires portal projectiles instead of instant-hit portals. See my personal page computergod666 for more info. You get the UTPortal gun (the one I scripted) later in the game to help with some of the advanced puzzles. Toward the end of the course, you are supposed to use the gravity gun and portal guns to fight your way past a few monsters. Then, things go wrong, the monsters smash their way out of the course, and you are left to escape on your own. The computer voice that has been guiding you is now disabled, and you must use your own initiative to solve any puzzles you encounter. Toward the end of the game, you are teleported to an alien planet as the monsters try to escape from the Earth. You are left with only an enhanced gravity gun and the classic portal gun. The game ends when you use your portal gun to create a dimensional paradox that simultaneously closes the portal and teleports you back to earth.

Progress

So far, I have created all the necessary weapons. However, I have no idea how to make a portal that you can see through (I don't think that is actually possible in UT).
