< Legacy:Mapping For UTXMP

Look for XMPArtifactNode actor here:

NavigationPoint >> JumpDest >> JumpSpot >> GameObjective >> XMPArtifactNode

Actor properties

Team >> TeamIndex 
set to 0 for RED, 1 for BLUE, 255 for neutral (green).
XMPArtifactNode >> HolderClasses 
add two artifacts in two slots. Make sure your Red and Blue node slots are different: for example, if you added artifacts to slots 0 and 2 in Red node, in Blue node add them to 1 and 3.
